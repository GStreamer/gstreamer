# Running the example

To run the iOS GstPlay example, you need to download and extract our GStreamer .xcframework package first.
You can get it [here](https://gstreamer.freedesktop.org/download/#ios).

Please extract the downloaded archive to this directory, so that the `GStreamer.xcframework` folder
is placed next to `GstPlay.xcodeproj`. The project is set up to detect it there automatically.
With that in place, you should be able to open the example in Xcode and run it in a simulator
or on your own device.

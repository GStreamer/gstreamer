## Details

Heap-based buffer overflow and out-of-bounds write in the RTP QDM2 depayloader.

## Impact

It is possible for a malicious third party to trigger a heap overflow or
out-of-bounds write that can result in a crash of the application, possibly
even remote execution.

## Solution

The gst-plugins-good 1.28.1 release addresses the issue. People using older
versions of GStreamer should apply the patch and recompile.

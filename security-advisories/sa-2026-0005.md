## Details

An out-of-bounds write in the RealMedia demuxer that can cause crashes for
certain input files.

## Impact

It is possible for a malicious third party to trigger out-of-bounds writes to
heap memory, which can result in a crash of the application.

## Solution

The gst-plugins-ugly 1.28.1 release addresses the issue. People using older
versions of GStreamer should apply the patch and recompile.

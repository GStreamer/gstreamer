## Details

An integer overflow in the RIFF parser that can cause crashes for
certain input files.

## Impact

It is possible for a malicious third party to trigger an integer overflow that
can result in out-of-bounds reads and writes to heap memory, and a crash of the
application.

## Solution

The gst-plugins-base 1.28.1 release addresses the issue. People using older
versions of GStreamer should apply the patch and recompile.

| IDs               | GStreamer-SA-2025-0009<br/>CVE-2025-67326<br/>CVE-2025-67327 |

## Details

Multiple out-of-bounds reads in the MIDI parser that can cause crashes for
certain input files.

## Impact

It is possible for a malicious third party to trigger an out-of-bounds read
that can result in a crash of the application.

## Solution

The gst-plugins-bad 1.26.10 release addresses the issue. People using older
versions of GStreamer should apply the patch and recompile.
